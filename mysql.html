MySQL

MySQL is een open source SQL-relationele databasebeheersysteem.

Een database is niet meer dan een georganiseerde verzameling gegevens die zo gestructureerd is dat die gegevens gemakkelijk te gebruiken en terug te vinden zijn

In een relationele database zijn je gegevens onderverdeeld in verschillende tabellen inplaats van dat ze alles in een opslagruimte gooien.
Daardoor maak je geen fouten als je bijvoorbeeld twee dezelfde namen hebben.

Veel high-traffic sites zoals Facebook en LinkedIn gebruiken MySQL

SQL is de programmeertaal waarmee je toegang kan krijgen tot de database
Hoe je een database maakt en SQL gebruikt leer je nog.
