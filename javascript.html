Javascript

JavaScript is een scripttaal waarmee je webpagina’s interactief kan maken. Een stukje van een scripttaal bestaat uit een aantal opdrachten die de browser uitvoert.
De browser ontvangt deze opdrachten als een HTML-pagina wordt ingeladen.

In deze opleiding leer je dus hoe je je website interactief maakt met Javascript, er zijn andere scripttalen dan javascript.
Het grote voordeel aan JavaScript is echter dat het in alle browsers wordt ondersteund.

Zie hieronder alle code dat ik geschreven heb doordat ik de opdracht niet las

Je voegt je code toe door
  <script>
    alert('Beep beep');
  </script>

in je code te plaatsen of als je een external file wilt linken:
<script src='path-to-javascript-file.js'></script>


Je script plaats je net voor de closing body tag zodat je website sneller laadt,
ook kan je 'defer' in je script zetten als het external is
Je script laadt dan parallel en wordt pas geëxecuteerd als je webpagina klaar is met laden.

Single line comments: // Dit is een single line comment
Multi line comments: /* Dit is een multi line comment */

Variabelen

Er zijn vier manieren om variabelen te verklaren:

  • Automatisch
  • var
  • let
  • const

variabelen worden automatisch verklaard als je het voor het eerst gebruikt,
maar het is het best als je let of const gebruikt

    When to Use var, let, or const?

  1. Always declare variables
  2. Always use const if the value should not be changed
  3. Always use const if the type should not be changed (Arrays and Objects)
  4. Only use let if you can't use const
  5. Only use var if you MUST support old browsers.

Conditionals

Een condiional gebruik je als je wilt weten of iets True of False is.
Een veelgebruikte conditional is if...else

De code die tussen if() staat wordt berekend/ bekeken, als de bewering/ berekening True is dan wordt de code die tussen de curley braces zit uitgevoerd, als het False is dan wordt de code van else uitgevoerd.

let iceCream = "chocolate";
if (iceCream === "chocolate") {
  alert("Yay, I love chocolate ice cream!");
} else {
  alert("Awwww, but chocolate is my favorite…"); }

Functies

Functies zijn stukjes code die buiten de rest van de script worden geschreven. Die code worden alleen maar uitgevoerd wanneer de script er expliciet om vraagt. Als een script veel dezelfde code bevat kun je de code die altijd hetzelfde is apart zetten in een functie en die functie telkens opnieuw oproepen.


Functies nemen vaak argumenten binnen de haakjes, je kan meerderen argumenten toevoegen als je ze aparthoudt met een komma


Je kan ook je eigen functies maken: je begint eerst met het woord function te typen; dit geeft aan dat je een functie wilt aanmaken. Vervolgens geef je een naam aan de functie die kort omschrijft wat ze doet.
Een functie oproepen doe je gewoon door de naam van de functie te typen met de () en ; erachter.



Met de onderstaande code print je "Hallo Wereld".

function hoihallo(){
 print("Hallo Wereld!");
}

hoihallo();
