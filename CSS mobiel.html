CSS

Wat is css nou eigenlijk? CSS staat voor Cascading Style Sheets. Afgekort CSS is een programeertaal die zorgt voor de opmaak van HTML bestanden. Het geeft de webpagina kleur, lettertype en algemene alle dingen die nodig zijn om een webpagina nodig moet hebben. CSS is de beste vriend van het maken van een HTMl bestand. CSS wordt dus vak gebruikt als je een HTML bestand wilt maken.

Hoe werkt css

Css werkt op verschillende manieren. Css is meestal gebruikt als een stylesheet voor allelei websites. In css kan je bijvoorbeeld lettertypes, achtergrond kleur, teksten verplaatsen ect.

Css is erg belangrijk, vooral als je het samen wilt koppelen aan HTML.

Css is een belangrijk middel als je met HTML wilt gaan werken. Css zorgt voor verschillende mogelijkheden die je kunt hebben op een pagina

Opbouw

De opbouw van css is best moeilijk voor sommige mensen, maar als je de hang ervan weet dan is het best wel te doen
