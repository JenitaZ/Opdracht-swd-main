UI&UX DESIGN

UI

UI&UX wat betekenen ze nou? UI staat voor 'User Interface',ook wel Gebruikersinterface in het Nederlands genoemd. Het heeft te maken met de gebruikers ervaring en gebruik van het website.

Wanneer je een website opzoek en erop klink of een mobiele app opstart is alles wat je ziet is dus User Interface. Het is dus soort van de 'voorkant' van een website.

Het is verstandig om de User Interface simpel en netjes te houden. Dat zorgt ervoor dat de gebruiker alles kan bijhouden en zo beter in de website kan nagiveren.

Een UX designer moet ervoor zorgen dat de website goed te gebruiken is voor de gebruiker.

Ze ontwerpen knoppen, menu's, icoontjes en de lay-out om ervoor te zorgen dat alles er aantrekkelijk uitziet

UX en UX hebben overeenkomsten, maar dat betekent niet dat ze hetzelfde zijn.

UX en UI ondersteunen elkaar, maar dat betekent niet dat ze hetzelfde zijn. Ze hebben allebei verschillende functies.

In de volgende alinea gaan we het hebben over wat UX is en hoe het werkt.

UX

UX staat voor User Experience, ook wel Gebruikerservaring in het Nederlands genoemd.

UX net zoals UI is het opbouw van een website en hoe het eruit ziet. Een groot verschil tussen hun is wel dat de UX meer gericht is naar de gebruikers gebruik van de website.

De gebruikerervaring moet ervoor zorgen dat de gebruiker een leuke ervaing heeft met hun website. Dus bijvoorbeeld een duidelijke zoekbar, een overzichtelijke linkjes.

Een UX designer heeft een duidelijke beeld van de doelgroep van wie zij de wesbite zit te maken. gebruiken ze wireframes om hun idee tot werkelijkheid te bringen

UI en UX designers hebben verschillen tussen elkaar. Deze verschillen zijn : De UX designers doen aan het creeren en maken van het uiterlijk van een website,

de UI designer kan nu de bouwtekening gebruiken om het ontwerp naadloos te laten aansluiten op je doelgroep en je bedrijf. Een UX en UI designer moeten dus goed met elkaar gaan werken
